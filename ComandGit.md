
## Ve a la carpeta del proyecto que clonaste


## Verifica la rama en la que estás

```bash
git branch
```

## Trae los últimos cambios del repositorio remoto

```bash
git pull origin master
```

## Si has hecho cambios locales

### Verifica la rama en la que estás

```bash
git branch
```

## Crea y cambia a la nueva rama

```bash
git checkout -b nombre-de-tu-rama
```



## Realiza tus cambios y haz commits normalmente

```bash
git add .
git commit -m "Descripción clara de los cambios realizados"
```
## Sube tu rama al repositorio remoto

```bash
git push origin nombre-de-tu-rama
```
# Para cambiarte entre ramas en Git, simplemente usas el comando:

```bash
git checkout nombre-de-la-rama
```
